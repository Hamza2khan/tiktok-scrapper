Location-Based Image Recognition

ML project using ResNet50 and Google Maps API to identify images based on location metadata. Achieved 75% accuracy on test data.

Live Link

Project Overview

This project focuses on training a convolutional neural network (ResNet50) to recognize locations using images' geo-metadata and visual cues. It's a hybrid system combining machine learning and real-world geolocation tools like Google Maps API.

The model was trained on a dataset of images tagged with location info and tested with real-world user uploads. It achieved a reliable accuracy of 75%, making it suitable for initial geo-verification tasks in security and travel applications.
